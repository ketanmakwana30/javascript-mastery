localStorage

~ 5MB Data Store

localStorage is a Browser Storage - it save data
  • Store
    • localStorage.setItem("name", "Ketan");

  • Get (Fetch Data)
    • let val = localStorage.getItem("name");

  • Remove
    • localStorage.removeItem("name");

  • Update
    • localStorage.setItem("name", "Krishna");

  • clear
    • localStorage.clear();

    - setItem are use for set and update the item , when item are not available so setItem make new item and all ready available to overright and make changes.
