Events

User Intigration Events

Mouse Event:

click, dblclick, mousedown, mouseup, mouseover, mouseout, mouseenter, mouseleave, contextmenu

Keyboard Event:

keydown, keyup, keypress (legacy)

Focus Event:

focus, blur (these do not propagate/bubble)

Document and Window Events

Lifecycle:

load (fires when page/resources finish loading), unload, beforeunload. Resize/Scroll: resize, scroll.

Error:

error (fires when resources like images or scripts fail to load).

Form and Input Events

Submission:

submit, reset.

Input:

input (fires on every change), change (fires on blur for most elements, or immediately for select tag), focus, blur.
